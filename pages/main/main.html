单页路由的说明

第一次打开首页,便会加载 main 模块, 模块需要通过define定义, 一个js只能有一个define. 默认main是在pages/main/目录下, 一个main模块,包含 main.html(模板), main.js (业务逻辑) ,如果需要修改路径,则通过 router.map 方法修改. 其它保持跟多页的开发方式保持一致.

load 页面跳转,更多参数用法请参考 bui.router 的 api

示例:
bui.load({ url: "pages/page2/page2.html ", param: {} }); 或者 router.load({ url: "pages/page2/page2.html ", param: {} });

back 页面后退,支持后退多层及回调

示例:
bui.back(); 或者 router.back();

refresh 页面刷新

示例:
bui.refresh(); 或者 router.refresh();

getPageParams 获取页面参数

示例:
bui.getPageParams(); 或者 router.getPageParams();

replace 替换当前页面,之前的历史记录会丢弃

示例:
bui.load({ url: "pages/page2/page2.html ", param: {}, replace: true }); 或者 router.replace({ url: "pages/page2/page2.html ", param: {} });
